Robot Dog Project

overview:
This project is a simple robot dog controlled by a Raspberry Pi 4. It uses 8 MG90 servo motors to move the legs and a servo controller board to manage all the servos.

Parts Used
- Raspberry Pi 4
- 8 MG90 servo motors
- Servo controller board
- Jumper wires
- Power supply for servos
- Robot dog frame/body

Future Improvements
- Add walking patterns
- Add a camera
- Add obstacle detection
